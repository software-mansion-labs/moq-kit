# Run iOS Demo

## Quick run

```bash
mise run run-ios
```

Builds and launches the subscriber app by default.

To run the publisher app instead:

```bash
mise run run-ios --app publisher
```

To target a simulator instead:

```bash
mise run run-ios --simulator
```
