# iOS SDK Rules

## Package Structure
- `ios/Package.swift` — Swift package with binary target `moqFFI` (the XCFramework)
- `ios/Sources/MoQKitFFI/moq.swift` — auto-generated UniFFI bindings (never edit)
- `ios/Frameworks/moqffi.xcframework` — git-ignored, generated by `mise run ios:ffi`

## Conventions
- `MoQSession` is `@MainActor`
- Use `try?` on `close()` calls to swallow errors during cleanup
- Error types: `MoQSessionError` (high-level), `MoqError` (low-level UniFFI — note lowercase 'q')
- When in doubt about UniFFI types/fields, read the generated `moq.swift` — don't guess
